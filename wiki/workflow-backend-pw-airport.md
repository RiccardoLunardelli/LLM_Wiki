# Workflow backend PW-Airport

## Sintesi

Descrizione del workflow backend PW-Airport.

## Dettagli

- Il workflow backend PW-Airport è un sistema di gestione dei voli che utilizza Unity e Python per la sua implementazione.
- Il sistema si compone di cinque fasi principali: startup e handshake di setup, logica di generazione dei path, bootstrap dello spawn iniziale, scheduling del lifecycle dei voli e gestione degli eventi runtime.
